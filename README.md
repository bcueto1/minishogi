# box-take-home
Box Take-Home-Exercise

To Compile In Interactive Mode:
```sh
$ javac MiniShogi.java
$ java MiniShogi.java
```

To Compile In File Mode:
```sh
$ cd MiniShogi
$ javac MiniShogi.java
$ java MiniShogi.java file.in
```

Noticeable Issues:
Not all available moves in check are shown correctly.
